🚀 QUICK START - GET GOING IN 5 MINUTES
═══════════════════════════════════════════════════════════════════════════════

1️⃣  Install dependencies
    npm run install:all

2️⃣  Configure environment
    Create backend/.env with MONGODB_URI and JWT_SECRET
    Create frontend/.env.development with REACT_APP_API_URL

3️⃣  Start services (3 terminals)
    Terminal 1: cd backend && npm run dev
    Terminal 2: cd frontend && npm start
    Terminal 3: ngrok http 5000 (optional)

4️⃣  Run tests
    npm run test --prefix backend
    npm run test --prefix frontend

5️⃣  View results
    Coverage: backend/coverage/lcov-report/index.html
    E2E: npm run cypress:open --prefix frontend
    Performance: npm run perf:load --prefix backend
