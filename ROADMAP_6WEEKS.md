## 📞 BLOCKERS & RISKS

### Critical Path Blockers

| Blocker | Impact | Mitigation |
|---------|--------|-----------|
| **Product photos** | PRIORITÉ 1 stuck | User to send ASAP |
| **Backend API** | PRIORITÉ 2 stuck | Must be accessible from Vercel |
| **Team bandwidth** | Can't parallelize | Consider 2-person team for faster delivery |

### Technical Risks

| Risk | Severity | Mitigation |
|------|----------|-----------|
| **Auth token loss on migration** | HIGH | Use httpOnly cookies + refresh token |
| **CORS issues** | MEDIUM | Setup API routes as proxy |
| **Build time explosion** | MEDIUM | Use ISR (revalidate: 3600) not full SSG |
| **Component incompatibilities** | MEDIUM | Test React components incrementally |
